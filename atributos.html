ATRIBUTOS

# First Last
1 id="texto" Establece un indicador único a cada elemento
2 class="texto" Establece la clase CSS que se aplica a los estilos del elemento
3 style="texto" Aplica de forma directa los estilos CSS de un elemento
4 title="texto" Establece el título del elemento (Mejora la accesibilidad)
5 ang="codigo" Indica el idioma del elemento
6 xml:lang="codigo" Indica el idioma del elemento, aunque tiene más prioridad que el atributo anterior y es obligatorio si se incluye el atributo lang
7 dir Indica la dirección del texto
8 onclick Ejecuta la acción cuando se realiza un clic sobre el elemento
9 ondblclick Ejecuta la acción cuando se realiza un doble clic sobre el elemento
10 onmousedown Ejecuta la acción cuando se detecta el botón pulsado del ratón
