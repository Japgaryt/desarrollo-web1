DESARROLLO WEB

ELEMENTOS

# First Last
1 html Define que el documento esta bajo el estandar de HTML 5
2 Head Representa una colección de metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo.
3 Tittle Define el título del documento, el cual se muestra en la barra de título del navegador o en las pestañas de página. Solamente puede contener texto y cualquier otra etiqueta contenida no será interpretada
4 Base Define la URL base para las URLs relativas en la página.
5 Link Usada para enlazar JavaScript y CSS externos con el documento HTML actual.
6 Meta Define los metadatos que no pueden ser definidos usando otro elemento HTML.
7 style Etiqueta de estilo usada para escribir CSS en línea.
8 script Define ya sea un script interno o un enlace hacia un script externo. El lenguaje de programación es JavaScript
9 noscript Define un contenido alternativo a mostrar cuando el navegador no soporta scripting.
10 body Representa el contenido principal de un documento HTML. Solo hay un elemento body en un documento.
